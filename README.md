# Binary Search Tree with unique_ptr

Реализация дерева поиска с использованием умных указателей.
Проект поддерживает безопасную модель владения, соответствие принципам RAII и корректную работу базовых операций BST

---

## Возможности

- Вставка элемента
- Поиск минимума
- Поиск преемника и предшественника
- Обход дерева
- Хранение структуры дерева через unique_ptr
- Автоматическое освобождение памяти
- Запрет копирования и разрешение перемещения (std::move)
> **Удаление узла (`delete_node`) намеренно не реализовано.**  
> Основная цель проекта — продемонстрировать корректную работу с `std::unique_ptr`, безопасное владение ресурсами и базовую структуру BST.

---

## Инвариант BST
Для каждого узла верно:
- значения в левом поддереве строго меньше value
- значения в правом поддереве больше или равны value
